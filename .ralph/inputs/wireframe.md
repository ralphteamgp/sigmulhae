# 🌿 식물식물해 — 웹앱 와이어프레임 v2.0

> 랩탑 전용 Web 애플리케이션 | 유저 스토리 v2.3 기반

---

## 전체 플로우 구조

사이드바 네비게이션 없이 **선형 스텝 방식**으로 진행됩니다.
첫 진입 화면이 허브 역할을 하며, 분석 전/후 상태에 따라 같은 자리의 콘텐츠가 바뀝니다.

```
┌──────────────────────────────────────────────────────────┐
│  TOPNAV  🌿 식물식물해                                      │
├──────────────────────────────────────────────────────────┤
│                                                          │
│  STEP 1  첫 진입 화면 (홈)                                 │
│          ↓ [우리 집 분석하기] 버튼 클릭                     │
│  STEP 2  주소 입력 + 사진 업로드                           │
│          ↓ [채광 분석 시작하기] 버튼 클릭                   │
│  STEP 3  분석 중 화면 (로딩)                               │
│          ↓ 분석 완료 시 자동 전환                           │
│  STEP 1' 첫 진입 화면 (홈) — 하단에 결과 노출              │
│          ↓ [식물 추천받기] 버튼 클릭                        │
│  STEP 4  식물 추천 리스트                                  │
│          ↓ [배치 추천 보기] 버튼 클릭                       │
│  STEP 5  식물 배치 추천                                    │
│          ↓ [케어 시작] 버튼 클릭                            │
│  STEP 6  식물 상태 코멘트 (케어)                           │
│          ↓ [케어 기록 보기] 버튼 클릭                       │
│  STEP 7  케어 기록 & 성장 트래킹                           │
│                                                          │
└──────────────────────────────────────────────────────────┘
```

|------|-----------|
| 로고 잎사귀 | 좌우 흔들림 (sway, 4s loop) |

| 화면 전환 | 페이드인 + translateY(10px→0) (0.28s) |
