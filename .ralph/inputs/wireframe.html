STEP 1 — 앱 첫 실행 / 홈 화면

PHASE 1 · 진입 & 온보딩

📖 유저 스토리

지수가 앱을 처음 열면 "내 공간에 맞는 식물 찾아줄게요" 카피와 함께 시작 화면이 노출되며, 별도 로그인 없이 바로 시작 가능하다. 첫 진입 시 분석 데이터가 없으므로 결과 영역에 온보딩 카피를 노출하여 분석을 유도한다. 공간 분석 완료 후에는 동일 화면에서 결과를 확인할 수 있다.

⚡ 전제 조건

  • 앱 최초 실행 상태 (분석 데이터 없음)
  • 로그인/회원가입 불필요
  • 로컬 스토리지에 저장된 이전 분석 결과 없음
  • 인터넷 연결 여부와 무관하게 화면 렌더링 가능

✅ 완료 기준

  • 앱 실행 3초 이내 홈 화면 렌더링
  • 온보딩 카피 및 CTA 버튼 노출
  • 로그인 없이 "우리 집 분석하기" 버튼 탭 가능
  • 분석 완료 후 재진입 시 결과 카드 노출 (빈 상태 대체)
  • 하단 네비게이션 탭 정상 동작

STEP 2 — 공간 분석 방식 선택

PHASE 1 · 진입 & 온보딩

📖 유저 스토리

지수는 공간 분석을 위해 분석할 공간의 주소를 등록하거나, 필요에 따라 사진을 업로드하여 더 자세한 분석을 받을 수 있다. "우리 집 분석하기" 버튼 탭 후 2가지 분석 방법 중 선택한다.

❓ 미정: 사진 업로드 최대 장수 제한 결정 필요

  • 카메라/갤러리 권한 미허용 상태 가능
  • 위치 권한 미허용 상태 가능

✅ 완료 기준

  • 2가지 분석 방법 카드 노출 (사진 / 주소)
  • 각 방법 선택 시 해당 플로우로 이동
  • 방법별 설명 텍스트 노출
  • 뒤로가기 시 홈 화면 복귀

STEP 3-A — 사진 업로드 플로우

PHASE 2 · 공간 데이터 수집

📖 유저 스토리

지수는 갤러리에서 창문이 보이는 방 사진을 1~3장 선택하여 업로드한다. AI가 사진에서 창문 위치·방향·면적을 추정하고, "이 창문이 남향 맞나요?" 같은 확인 질문을 제시한다. 아니오 선택 시 나침반 앱 실행 또는 텍스트 입력으로 방위각을 수집한다.

❓ 미정: 나침반 앱 실행 vs 텍스트 입력 방식 결정 필요

⚡ 전제 조건

  • STEP 2에서 "사진 업로드" 선택
  • 갤러리 접근 권한 허용
  • 사진 1장 이상 업로드 완료
  • AI 분석 API 연결 상태

✅ 완료 기준

  • 사진 1~3장 업로드 UI 제공 (최대 장수 미정)
  • 업로드 후 AI 분석 결과로 창문 방향 추정값 노출
  • 확인 질문 (예/아니오) 응답 가능
  • "아니오" 선택 시 방위각 입력 화면 전환
  • 최종 방향 확인 후 STEP 4로 이동

STEP 3-B — GPS + 공공API 플로우

PHASE 2 · 공간 데이터 수집

📖 유저 스토리

지수는 분석하려는 공간의 주소를 텍스트로 입력하여 제출한다. 위치 권한 허용 시 GPS 좌표가 자동 수집되고, 건축물대장 공공 API로 건물 층수·면적·건축 연도가 자동 조회된다. 스트리트뷰 자동 연동으로 맞은편 건물 높이 등 장애물도 분석된다.

⚡ 전제 조건

  • STEP 2에서 "주소 입력" 선택
  • 위치 권한 허용 (선택적)
  • 건축물대장 공공 API 연동 완료
  • 스트리트뷰 API 연동 완료

✅ 완료 기준

  • 주소 검색 → 자동완성 결과 노출
  • 주소 선택 후 건물 정보 자동 조회 및 표시
  • 스트리트뷰 이미지 자동 로드
  • 층수 직접 입력 가능 (공공 API 미지원 시)
  • 정보 확인 후 STEP 4로 이동

STEP 6 — 식물 추천 리스트

PHASE 4 · 식물 추천

📖 유저 스토리

지수는 일조량 분석을 통해 공간에 적합한 식물을 추천받는다. 태그를 통해 추천된 식물을 직관적으로 파악할 수 있으며, 채광 등급 기반으로 공간별 3~5개 식물이 추천된다. 초보자 필터가 기본 ON 상태이다.

⚡ 전제 조건

  • STEP 5 채광 결과 데이터 존재
  • 식물 DB 연동 완료
  • 초보자 필터 기본값 ON

✅ 완료 기준

  • 공간별 식물 3~5개 카드 노출
  • 각 카드: 이름, 난이도, 추천 이유, 예상 효과 태그
  • "키우기 쉬운 것만 보기" 토글 기본 ON
  • 공간 탭 전환 가능 (거실/침실 등)

지수는 2D 평면도 위에서 추천 식물의 최적 배치 위치를 한눈에 파악한다. 햇빛 스캔 애니메이션 후 최적 지점에서 멈추며, 식물을 햇빛 구역에 놓으면 초록, 음지에 놓으면 빨강으로 표시된다. 하단/우측 패널에서 배치된 식물 목록을 관리할 수 있다.

⚡ 전제 조건

  • STEP 5 채광 결과 + 2D 평면도 데이터 존재
  • STEP 6에서 추천 식물 1개 이상 선택
  • 식물 아이콘 드래그 앤 드롭 기능 구현

✅ 완료 기준

  • 2D 평면도 + 햇빛 스캔 → 최적 지점 정지 애니메이션
  • 추천 식물 아이콘 평면도 위 배치 가능
  • 배치 위치에 따라 초록/빨강 피드백 표시
  • 식물 아이콘 삭제 가능 (롱프레스 or X 버튼)
  • 하단 패널에 배치된 식물 목록 + 관리 기능
  • 배치 텍스트 설명 노출 ("거실 창가 왼쪽이 딱 맞아요")
